## 💡 使用技巧

### 1. 包含空格的文字用引号

```
@机器人 /memegen drake "使用复杂的 AI API" "使用免费的 memegen.link"
```

### 2. 支持中文模板名

```
@机器人 /memegen 分心男友 Replicate Memegen
```

### 3. 可以只有上方文字

```
@机器人 /memegen drake "好的选择"
```

### 4. 可以不加文字（使用模板默认）

```
@机器人 /memegen drake
```

## 🆚 Memegen vs Milady Meme

| 特性 | Memegen | Milady Meme |
|------|---------|-------------|
| **模板数量** | 207个经典模板 | 自定义NFT |
| **费用** | 完全免费 | 免费 |
| **速度** | 秒出 | 快速 |
| **自定义** | 固定模板 | 完全自定义 |
| **适用场景** | 快速梗图 | 个性化梗图 |
| **命令** | `/memegen` | `/meme` |

## 🎯 什么时候用 Memegen？

✅ **适合使用 Memegen：**
- 需要快速生成经典梗图
- 使用大家都认识的模板
- 不需要个性化 NFT 元素

❌ **不适合使用 Memegen：**
- 需要 Milady NFT 风格
- 需要叠加多个图层
- 需要特定 NFT 角色
